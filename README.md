# Where Managers Waste Time – Interactive Simulation

Most managers are busy.  
But very few know where their time actually goes.

This project is a decision-based simulation that shows how managers spend their day—and what that means for team performance.

---

## 🎯 Problem

Managers spend excessive time in:
- Meetings  
- Reporting  
- Reactive tasks  

While under-investing in:
- Coaching  
- Strategy  
- Team development  

This leads to:
- Low productivity  
- High dependency on manager  
- Poor team growth  

---

## 💡 Solution

This tool simulates a manager’s workday.

Users:
- Make real-world decisions across 7 scenarios  
- Spend limited time (8 hours)  
- See how their choices impact time allocation  

---

## 🧠 What It Reveals

At the end of the simulation, users get:

- 📊 Time distribution (Meetings, Coaching, Strategy, etc.)  
- 🔴 Leadership pattern (Reactive vs Strategic)  
- 🟡 Business impact on team  
- 🟢 Practical actions to improve  

---

## ⚙️ Key Features

- Interactive decision-based gameplay  
- Realistic workplace scenarios  
- Time constraint (8-hour day simulation)  
- Built-in time allocation analyzer  
- Benchmark comparison (ideal vs actual)  
- No data collection — runs fully in browser  

---

## 🚀 Why This Is Different

- Not a quiz  
- Not a survey  
- Not a training module  

It is a **behavior simulation tool** that reflects real managerial decision-making.
